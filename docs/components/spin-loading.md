## SpinLoading 属性

| 属性名    | 说明                         | 类型                     | 可选值                             | 默认值 |
| --------- | ---------------------------- | ------------------------ | ---------------------------------- | ------ |
| className | 自定义样式类名               | string                   | -                                  | -      |
| style     | 自定义样式                   | `object` `CSSProperties` | -                                  | -      |
| loading      | 是否显示加载中              | `boolean`                                  | false      |
| tip      | 加载中提示文本              | `string`                                  | '加载中...'      |
