## Dialog 属性

| 属性名    | 说明                         | 类型                     | 可选值                             | 默认值 |
| --------- | ---------------------------- | ------------------------ | ---------------------------------- | ------ |
| activeTab | 当前路由               | string                   | -                                  | /      |
| tabs | 默认tabs列表 | array | [TabItem](./tabs#TabItem-配置) | - |

## TabItem 配置

| 属性名    | 说明     | 类型     | 可选值      | 默认值    |
| ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------- | --------------------------------------------------- | ----------------------------------------------------|
| name | 路由名称 | string | -  | - |
| title | 路由title | string | -  | - |
| path | 路由地址 | string | -  | - |
| meta | 路由meta | object | -  | - |
| query | 路由query | object | -  | - |
| params | 路由params | object | -  | - |
