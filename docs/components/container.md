# Container

方便统一设置容器样式。


## Container 属性

| 属性名     | 说明                      | 类型           | 可选值                  | 默认值       |
| ---------- | ------------------------ | -------------- | ---------------------- | ------------ |
| className | 自定义样式类名 | string | - | - |
| style | 自定义样式 | `object` `CSSProperties` | - | `padding: 15px` |
| scrollbar | 是否使用`ElScrollbar`组件嵌套 | boolean | - | false |
| card | 是否开启卡片形式 | boolean | - | false |
| padding | 直接设置容器内边距值 | string、number、boolean | - | true `padding: 15px` |
