## 使用包管理器
**安装及相关配置，建议使用[pnpm](https://www.pnpm.cn/)安装**

```shell
# node版本建议>=16 pnpm >= 8.X.X
# pnpm
$ pnpm add next-element-vue

# Yarn
$ yarn add next-element-vue

# NPM
$ npm install next-element-vue --save
```
